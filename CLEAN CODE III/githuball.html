Início

é uma plataforma para hospedagem e colaboração em projetos de desenvolvimento de software, onde você pode compartilhar, colaborar e versionar código usando o sistema de controle de versão Git. É muito útil para projetos de programação, como o jogo que você criou, pois permite:.

Sobre

.GitHub usa o sistema Git para controle de versão, permitindo que você acompanhe mudanças no código ao longo do tempo. Com Git, você pode: Criar "commits" para registrar mudanças incrementais. Navegar e voltar para versões anteriores do seu código. Trabalhar com branches (ramificações), facilitando a colaboração e a experimentação sem impactar o código principal.

principais conceitos

Principais Conceitos Repositório (Repo): Um local onde o código-fonte do seu projeto é armazenado. É o ponto central do seu projeto. Commits: Cada alteração feita no repositório é registrada como um commit, com uma mensagem que descreve o que foi alterado. Branches: Permitem criar diferentes "versões" do projeto para trabalhar em paralelo, sem afetar a versão principal (geralmente chamada de main ou master). Pull Requests (PRs): São usados para propor mudanças em uma branch. Quando um PR é aprovado, ele pode ser "mergeado" (mesclado) na branch principal. Issues: Um sistema de rastreamento de problemas e sugestões para ajudar a gerenciar tarefas no projeto.
