Clean code & Modulação em java

Clean é um conceito de programação que se refere a práticas e princípios que tornam o código mais legível, prático e fácil de manter. Algumas características essenciais do código limpo incluem uma legibilidade. O código deve ser fácil de ler e entender, Isso envolve o uso de nomes descritivos para variáveis, funções e classes.

Modulação

refere-se à prática de dividir o código em partes menores, chamadas de módulos, que podem ser gerenciados, desenvolvidos e testados de forma independente. Essa abordagem traz várias vantagens, como uma organização do coódigo. Módulos ajudam a manter o código organizado, permitindo que os desenvolvedores separem diferentes funcionalidades em arquivos ou partes distintas
